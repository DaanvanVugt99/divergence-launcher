# Changelog

## 0.1.0

Initial launcher development release.

- Added Electron, React, TypeScript, Vite, Tailwind, and shadcn/ui shell.
- Added local ROM verification and xdelta patching.
- Added managed patched ROM export/open-folder actions.
- Added external mGBA detection, configuration, and launch.
- Added macOS arm64 packaging and artifact verification.
- Added GitHub Actions release artifact builds.
- Added optional Apple Developer ID signing and notarization support.
